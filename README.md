
# KP_01-3D-LiDAR

## Description
The KP_01 LiDAR is a very cost-efficient 3-dimensional LiDAR module with a brushless pan axis, and servo tilt axis. It integrates a TF-Luna ToF LiDAR and a 10-notch rotary encoder for precise angle and RPM sensing. The LiDAR runs from an ESP32, with communication of all points in simple x-y-z coordinates done over serial.

## Parts List
* Benewake TF-Luna ToF LiDAR sensor
* 9g micro servo
* Redhobby 2-4S AM32 ESC
* 6-channel Slip ring with hole
* Blue slot optocoupler
* 2208 90Kv Brushless gimbal motor
* ESP32 Dev module
* M6 Torx screws
* 5x10x4mm bearings
